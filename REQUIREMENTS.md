# Ember & Oak — Requirements

## Core Sections (Priority Order)

### P0 — Must Have
1. Hero with reservation CTA and atmosphere preview
2. Menu with categories (Starters, Mains, Desserts, Drinks)
3. About the restaurant / Our Story
4. Hours & Location with map placeholder
5. Reservation / booking form
6. Contact info (phone, email, address)

### P1 — Should Have
1. Chef's specials / featured dishes
2. Photo gallery / ambiance showcase
3. Customer reviews / testimonials
4. Private events / catering section
5. Instagram feed placeholder

### P2 — Nice to Have
1. Gift cards section
2. Weekly specials / happy hour info
3. Blog / recipes section
4. Loyalty program mention

## Technical Requirements
- Single-page HTML/CSS/JS
- Dark theme by default (with light mode toggle)
- Mobile-first responsive
- Fast load, no frameworks
- Semantic HTML5, accessible
- Elegant scroll animations
